# Stichting Voedselbank voor Dieren 010 – Hulpverlenersportaal

Hulpverlenersportaal en chatapp voor Stichting Voedselbank voor Dieren 010.

## Functionaliteiten

* Hulpverlenersportaal
* Chatfunctionaliteit
* PWA ondersteuning
* Offline functionaliteit via service worker
* Mobiele installatie mogelijk
* Beveiligde toegang
* Responsief ontwerp

## Bestandsstructuur

* `index.html` – hoofdpagina
* `hulpverlenersportaal.html` – portaal voor hulpverleners
* `manifest.json` – PWA configuratie
* `service-worker.js` – offline caching
* `js/` – JavaScript bestanden
* `.htaccess` – server configuratie

## Technologieën

* HTML5
* JavaScript
* Progressive Web App (PWA)
* Service Workers
